Расширенный калькулятор

	История версий:

	Оконная версия:
	Разработана Майским Александром в августе 2021г.

	Консольная версия:
	Дополнена Майским Александром в январе 2020г.
	Переработана Бьярне Страуструпом в мае 2007г.
	Переработана Бьярне Страуструпом в августе 2006г.
	Переработана Бьярне Страуструпом в августе 2004г.
	Разработана Бьярне Страуструпом (bs@cs.tamu.edu) весной 2004г.

	Эта программа реализует основные выражения калькулятора.
	Ввод осуществляется из потока cin; вывод - в поток cout.

	Грамматика для ввода:

	Инструкция:
		Выражение		- математические операции с числами, 
					  переменными и константами, при возникновении
					  ошибки обозначается "primary"

		Добавление_переменной	- программа может хранить и использовать числа
					  под именами, которые вы выбираете

		Степень_числа		- "pow(x,y)", возводит x в степень y

		Корень_квадратный	- "sqrt(x)", вычисляет корень квадратный из x

		Помощь			- кнопка "информация", но вы уже это знаете...

		Выход			- для выхода введите "quit"

		Вывод в файл		- вы можете настроить программу на запись резултатов
					  в файл, если файла с таким именем не существует -
					  программа создаст его

		Ввод из файла		- вы можете настроить программу на ввод данных из файла,
					  рекомендуется предварительно подключить файл для вывода,
					  иначе результаты вычислений не будут выведены корректно
					  
					  Для возврата к обычному режиму требуется перезагрузить
					  программу
	Вывод в файл:
		to имя_файла.формат_файла

	Ввод из файла:
		from имя_файла.формат_файла

	Выход:
		quit

	Корень_квадратный:
		sqrt( Выражение )

	Степень_числа:
		pow( Выражение , Выражение )

	Добавление_переменной:
		# Имя_переменной = Выражение
		# const Имя_переменной = Выражение

	Имя_переменной:
		символы
		Имя_переменной '_'
		Имя_переменной цифры

	Выражение:
		Терм
		Выражение + Терм
		Выражение - Терм

	Терм:
		Первичное_выражение
		Терм * Первичное_выражение
		Терм / Первичное_выражение
		Терм % Первичное_выражение

	Первичное_выражение:
		Число
		Имя_переменной
		Имя_переменной = Выражение
		( Выражение )
		- Первичное_выражение

	Число:
		Литерал_с_плавающей_точкой

	Кроме этого, программа распознает римские числа,
	введенные латиницей в верхнем регистре.
